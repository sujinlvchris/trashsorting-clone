# Thailand Waste Sorter Clone

Static single-page clone of `https://trashsorting.vercel.app/`.

Public URL:

```text
https://sujinlvchris.github.io/trashsorting-clone/
```

Run it locally:

```bash
python3 -m http.server 4173
```

Then open `http://localhost:4173/`.

The page implements image upload, drag-and-drop, preview, loading state, and a four-bin sorting result. The classifier is a local demo classifier using filename keywords and basic image color signals, so it does not depend on the original private API.
